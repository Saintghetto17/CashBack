Уважаемые проверяющие!

Обратите внимание, что в Launcher я пишу и читаю файл data.bin
по относительному пути, чтоб это замечательнейшим образом работало
у вас на компьютере при запуске из Intelij-IDEA.
(Intelij->Edit_Configurations->Cli args->вставляете любую команду из папки commands)

Тем не менее процесc запуска из Intelij не автоматизирован и утомителен.
Поэтому в папке scripts лежит run.bat для запуска процесса. Там написана
последовательность команд запуска Launcher.java для того, чтоб
в моей базе был пример из задания. ОДНАКО! ЧТОБ автоматизированный процесс запустился, придется
в конструкторе FileInputStream/FileOutputStream поменять путь на абсолютный(не знаю почему, но иначе скрипт бросает
ошибку, что не может найти файл) :
Пример : C:\Users\ilyan\Desktop\LiptSoft\Cashback\src\main\java\bank\data.bin



Все данные, как вы и просили сериализуются в data.bin. Если вы хотите после
исполнения программы через run.bat поиграться самостоятельно, просто сотрите
содержимое data.bin либо ручками, либо запустив python script : delete_data.py

Все класс файлы я сохраняю в папку out, как и положено.
В папке libs лежит необходимая для удобной работы с CL библиотека picocli.

Для вашего удобства/неудобства (например если вы будете сразу из Intelij запускать) я в pom.xml уже добавил
эту библиотеку как зависимость.


Тесты находятся в папке test и проверяют функционал компонентов банка.
